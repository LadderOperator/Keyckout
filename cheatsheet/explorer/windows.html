• Win - 打开“开始”菜单。

  • Win + A - 打开操作中心。

  • Win + Alt + D - 在任务栏中打开日期和时间。

  • Win + Alt + 数字 (0-9) - 在任务栏中的数字位置打开应用的跳转列表。

  • Win + B - 在任务栏中设置焦点通知区域。

  • Win + C - 打开/关闭 Copilot。

  • Win + 逗号 (，) - 暂时查看桌面。

  • Win + Ctrl + D - 创建虚拟桌面。

  • Win + Ctrl + Enter - 打开“讲述人”。

  • Win + Ctrl + F - 在域网络上打开搜索设备。

  • Win + Ctrl + F4 - 关闭活动虚拟桌面。

  • Win + Ctrl + 向左键 - 切换到左侧的虚拟桌面。

  • Win + Ctrl + 数字 (0-9) - 切换到任务栏中数字位置中应用的最后一个活动窗口。

  • Win + Ctrl + Q - 打开快速助手。

  • Win + Ctrl + 向右键 - 切换到右侧的虚拟桌面。

  • Win + Ctrl + Shift + B - 黑屏或空白屏幕时唤醒设备。

  • Win + Ctrl + Shift + 数字 (0-9) - 在任务栏中的数字位置以应用管理员身份打开另一个实例。

  • Win + Ctrl + 空格键 - 更改以前的选定输入选项。

  • Win + D - 显示和隐藏桌面。

  • Win + 向下键 - 最小化应用窗口。

  • Win + E - 打开文件资源管理器。

  • Win + Esc - 退出放大镜。

  • Win + F - 启动反馈中心应用。

  • Win + 正斜杠 (/) - 启动 IME 重新转换。

  • Win + G - 启动游戏栏应用。

  • Win + H - 打开听写功能。

  • Win + 主页 - 最小化或最大化除活动桌面窗口外的所有窗口。

  • Win + I - 打开设置。

  • Win + J - 在 Windows 10 的提示上设置焦点（如果适用）。

  • Win + K - 打开连接设置。

  • Win + L - 锁定计算机。

  • Win + 向左键 - 向左对齐应用或窗口。

  • Win + M - 最小化所有窗口。

  • Win + 减 ( ) - 使用放大镜缩小字体功能。

  • Win + 数字 (0-9) - 在任务栏中的编号位置打开应用。

  • Win + O - 锁定设备方向。

  • Win + P - 打开项目设置。

  • Win + 暂停 - 显示系统属性对话框。

  • Win + 句点 (.) 或分号 (;)   - 打开表情符号面板。

  • Win + 加号 (+) - 使用放大镜放大。

  • Win + PrtScn - 捕获“屏幕截图”文件夹中的完整屏幕截图。

  • Win + R - 打开“运行”命令。

  • Win + 向右键 - 向右贴靠应用或窗口。

  • Win + S (或 Q) - 打开搜索。

  • Win + Shift + 向下键 - Windows 11： 如果窗口已贴靠或最大化，则还原窗口。 Windows 10： 在保持宽度的同时，垂直最大化或最小化活动窗口。

  • Win + Shift + 向左键 - 移动活动窗口以在左侧进行监视。

  • Win + Shift + M - 还原桌面上最小化的窗口。

  • Win + Shift + 数字 (0-9) - 在任务栏中的数字位置打开另一个应用实例。

  • Win + Shift + 向右键 - 移动活动窗口以在右侧监视。

  • Win + Shift + S - 创建屏幕屏幕截图的一部分。

  • Win + Shift + 向上键 - 将桌面窗口拉伸到屏幕顶部和底部。

  • Win + Shift + V - 循环浏览通知。

  • Win + 空格键 - 更改键盘布局和输入语言。

  • Win + T - 在任务栏中循环浏览应用。

  • Win + 选项卡 - 打开任务视图。

  • Win + U - 打开“轻松访问”设置。

  • Win + 向上键 - 最大化应用窗口。

  • Win + V - 打开剪贴板箱。

  • Win + W - 打开 Windows Ink 工作区。

  • Win + X - 打开快速链接菜单。

  • Win + Y - 更改桌面和混合现实之间的输入。

  • Win + Z - 在桌面体验和 Windows Mixed Reality 之间切换输入。
